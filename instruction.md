MyMeet - Architecture de la Plateforme de Visioconférence
Note : Cette plateforme est une application de visioconférence standard, identique à Google Meet,
Zoom, ou WhatsApp Appel de Groupe. MyMine n’est pas intégré ici. C’est un banc de test
autonome sur lequel MyMine viendra s’intégrer ultérieurement via une API d’extension.

Pourquoi P2P mesh pour l’instant :

• Aucun serveur média complexe à gérer

Passage à SFU (Mediasoup) en v2 si vous dépassez 4-6 participants régulièrement.

10. Résumé
MyMeet est une plateforme de visioconférence standard et autonome, fonctionnellement équiva-
lente à Google Meet ou Zoom. Elle est construite sur :
• React + Vite pour le frontend
• Node.js + Express + Socket.io pour le backend et le signaling
• WebRTC P2P pour la communication média
• PostgreSQL + Redis pour la persistance et le cache
Elle sert de banc de test avant l’intégration de MyMine, qui viendra s’y greffer comme un module
d’extension sans altérer le fonctionnement standard de la visioconférence.
